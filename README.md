ウェブページ上で選択したテキストについて AI に質問し、回答をページ内にハイライト保存できるブラウザ拡張機能です。チャット履歴を汚さずにサイドメモとして使えます。

**対応ブラウザ**：現在は Chrome / Chromium 系ブラウザのみ対応しています。

## 使い方

### インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を ON にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

### 初期設定

1. ブラウザ右上の拡張機能アイコンをクリック
2. AIプロバイダーを選択（デフォルト: Gemini）
3. モデルを選択
4. APIキーを入力して「保存」

### 基本操作

1. ウェブページ上でテキストを選択
2. 表示される「💬 Ask」ボタンをクリック
3. 質問を入力して送信（`Ctrl + Enter` でも送信可）
4. 回答が表示され、選択テキストが黄色くハイライトされる
5. ハイライトをクリックすると保存した回答をいつでも確認できる

Q&A 履歴は拡張機能のポップアップ画面から確認・削除できます。

## トークンについて

LLM の利用コストは「トークン」単位で計算されます。トークンとは単語や文字の断片で、日本語は英語より多くのトークンを消費します。

**目安：**
- 日本語 1 文字 ≈ 1〜2 トークン
- 英語 1 単語 ≈ 1〜2 トークン

**1 回の質問で消費するトークン（目安）：**

| 内容 | トークン数 |
|------|-----------|
| システムプロンプト（固定） | 約 80 |
| 選択テキスト（100文字程度） | 約 100〜200 |
| 質問文（30文字程度） | 約 30〜60 |
| 回答（200文字程度） | 約 200〜400 |
| **合計** | **約 400〜740** |

Claude Sonnet 4.6 で 1 回あたり約 **$0.003〜0.01**（1 円未満）程度です。
Gemini 無料枠であれば 1 日 25 回（2.5 Pro）まで無料で使えます。

## セキュリティ

- API キーは Chrome の `storage.local` に保存され、ローカルのみで管理されます
- API 通信は `background.js` 経由で行われ、APIキーはコンテンツスクリプトに露出しません
- HTML エスケープ処理により XSS を防止しています
